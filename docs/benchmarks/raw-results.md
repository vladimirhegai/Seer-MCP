# Raw Results

Full indexing-performance numbers behind the [benchmark summary](../benchmarks.md).

Generated by:

```bash
npm run scale-test
```

Each repo is indexed once from scratch and then indexed again with every file
unchanged.

## Environment

| Item | Value |
|---|---|
| Node | v26.1.0 |
| Platform | Windows |
| Machine class | Consumer laptop |

Codebase-Memory is excluded from the public comparison set because it is a
self-validation fixture.

## Terms

| Term | Meaning |
|---|---|
| Fresh index | Full index build for the repo. |
| Cached re-index | Second pass with unchanged files. |
| ms/file | Fresh index time divided by indexed files. |
| Cache speedup | Fresh index time divided by cached re-index time. |

## Reproduce

```bash
npm run scale-test
npm run scale-test -- --skip cbm
```
